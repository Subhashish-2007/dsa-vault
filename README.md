# DSA Vault

DSA Vault is a coding practice tracker that helps users save, organize, and track problems from multiple competitive programming and interview platforms in one place.

The project allows users to create custom collections, add problem links, detect the problem platform, mark questions as solved, and monitor progress through a clean dashboard. It is designed for learners who want to manage their DSA practice in a structured way instead of switching between different platforms and losing track of solved questions.

Currently, DSA Vault stores data locally in the browser and also supports export/import backups so users can move their saved questions when needed.

## Future Plans

In the future, DSA Vault can be expanded into a more powerful practice platform with:

- Syncing problems directly from platforms like LeetCode, Codeforces, CodeChef, and AtCoder
- User accounts and cloud-based storage
- Automatic progress tracking from connected coding profiles
- Difficulty tags and topic-wise filters
- Search and sorting for saved problems
- Revision reminders
- Notes for each problem
- Daily practice goals and streak tracking
- Public profiles and shareable practice collections
- Analytics for solved problems, weak topics, and platform-wise progress
